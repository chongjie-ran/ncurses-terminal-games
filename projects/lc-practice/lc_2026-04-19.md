## 03:23 DP专题二 - 股票IV/背包/整数拆分

### LC188 股票买卖IV (Hard) - k次交易
**核心**: 三维DP降维 + 贪心剪枝优化
**关键洞察**: k >= n/2 → 退化为无限次交易（贪心）
```cpp
if (k >= n/2) {
    // 贪心：累加所有上涨
    for (i=1;i<n;i++) if(prices[i]>prices[i-1]) profit+=prices[i]-prices[i-1];
}
```
**二维DP核心**:
- `cash[t][i]` = 第t次交易、第i天、不持有
- `hold[t][i]` = 第t次交易、第i天、持有
- `hold[t][i] = max(hold[t][i-1], cash[t-1][i-1] - prices[i])` ← 用上一次的cash买入
**验证**: ✅ [2,4,1]→2 / [3,2,6,5,0,3]→7 / 贪心优化验证→8

### LC983 最低票价 (Medium)
**核心**: 日期型DP，窗口式转移
- dp[i] = 到第i天的最低消费
- 转移：dp[i]=min(dp[i-1]+1天票, dp[i-7]+7天票, dp[i-30]+30天票)
**验证**: ✅ days=[1,4,6,7,8,20],costs=[2,7,15]→11

### LC474 一和零 (Medium) - 01背包
**核心**: 二维背包（两个维度m和n）
- dp[i][j] = i个0和j个1最多包含的字符串数
- 降维：反向遍历防止一物多选
**验证**: ✅ ["10","0001","111001","1","0"], m=5,n=3→4

### LC343 整数拆分 (Medium)
**核心**: 拆分后乘积最大化
- dp[i] = 整数i的最大乘积
- 转移：dp[i] = max(j*(i-j), j*dp[i-j]) for j in [1,i)
**验证**: ✅ n=2→1 / n=10→36
