# LC练习 - 位运算专题

## 核心模式

### 1. 基本位操作
| 操作 | 表达式 | 说明 |
|------|--------|------|
| 获取最低位1 | `x & (-x)` | 提取最右侧的1 |
| 清除最低位1 | `x & (x - 1)` | 消除最右侧的1 |
| 获取最低位0 | `~x & (x + 1)` | 提取最右侧的0 |
| 判断是否为2的幂 | `x > 0 && (x & (x - 1)) == 0` | 只有一位为1 |

### 2. 常用技巧
| 场景 | 技巧 |
|------|------|
| 统计1的个数 | Brian Kernighan: `while(x) { x &= x-1; cnt++; }` |
| 交换两数 | `a ^= b; b ^= a; a ^= b;` |
| 找出唯一数 | `全员异或` 后再 `x ^ x = 0` |
| 找出两个唯一数 | 分组异或 + `x & (-x)` |

### 3. 异或性质
- `a ^ a = 0`
- `a ^ 0 = a`
- `a ^ b ^ c = a ^ (b ^ c) = (a ^ b) ^ c`
- `a & ~a = 0`
- `a | ~a = 全1`

## 题目练习

### LC136 - 只出现一次的数字
**问题**: 数组中除一个元素外其余都出现两次，找出那个唯一的数
**核心**: 全员异或，`ans ^= nums[i]`
**验证**: [2,2,1]→1 ✓, [4,1,2,1,2]→4 ✓

### LC137 - 只出现一次的数字 II
**问题**: 其余数字都出现三次，找出那个唯一的数
**核心**: 位计数法，每位模3
**复杂度**: O(32n) 逐位统计

### LC260 - 只出现一次的数字 III
**问题**: 两个数各出现一次，其他都出现两次
**核心**: 
1. 全员异或得到 `x = a ^ b`
2. `diff = x & (-x)` 提取最低位1
3. 按该位分组异或

### LC191 - 位1的个数
**问题**: 无符号整数二进制中1的个数
**核心**: Brian Kernighan算法
```cpp
int hammingWeight(uint32_t n) {
    int cnt = 0;
    while (n) {
        n &= (n - 1);
        cnt++;
    }
    return cnt;
}
```

### LC231 - 2的幂
**问题**: 判断n是否为2的幂
**核心**: `n > 0 && (n & (n - 1)) == 0`
**验证**: 1→T, 16→T, 3→F ✓

### LC190 - 颠倒位
**问题**: 颠倒32位无符号整数的位顺序
**核心**: 分组交换
```cpp
uint32_t reverseBits(uint32_t n) {
    n = (n >> 16) | (n << 16);
    n = ((n & 0xff00ff00) >> 8) | ((n & 0x00ff00ff) << 8);
    n = ((n & 0xf0f0f0f0) >> 4) | ((n & 0x0f0f0f0f) << 4);
    n = ((n & 0xcccccccc) >> 2) | ((n & 0x33333333) << 2);
    n = ((n & 0xaaaaaaaa) >> 1) | ((n & 0x55555555) << 1);
    return n;
}
```

### LC201 - 数字范围按位与
**问题**: 范围[m,n]内所有数字按位与的结果
**核心**: 找公共前缀，Brian Kernighan消除最低位1
```cpp
int rangeBitwiseAnd(int m, int n) {
    while (m < n) {
        n &= (n - 1);
    }
    return n;
}
```

## 模式要点

1. **提取最低位1**: `x & (-x)` - 关键技巧
2. **消除最低位1**: `x &= (x - 1)` - 统计1的个数
3. **唯一数问题**: 
   - 一个：全员异或
   - 两个：分组异或 + diff
   - k个：位计数模k
4. **公共前缀**: 多次消除最低位1直到m<=n
