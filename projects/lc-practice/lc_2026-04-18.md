## 专题: 堆 (Heap) 深入

### LC23 合并K个升序链表 (Hard)
**问题**: K个有序链表合并成一个大有序链表
**核心**: 最小堆（priority_queue），每次取最小节点

```cpp
// 核心模式
priority_queue<pair<int, ListNode*>, vector<pair<int, ListNode*>>, greater<pair<int, ListNode*>>> pq;
for (auto& head : lists) {
    if (head) pq.emplace(head->val, head);
}
while (!pq.empty()) {
    auto [val, node] = pq.top(); pq.pop();
    if (node->next) pq.emplace(node->next->val, node->next);
}
```

**验证**: ✅ 4/4 cases passed

### LC295 数据流中位数 (Hard)
**问题**: 动态数据流求中位数
**核心**: 两个堆 - maxHeap存小的一半，minHeap存大的一半

```cpp
// 核心模式
priority_queue<int> maxHeap;  // 左边较小
priority_queue<int, vector<int>, greater<int>> minHeap;  // 右边较大
// 平衡: maxHeap.size() == minHeap.size() || maxHeap.size() == minHeap.size()+1
```

## 专题: 字符串算法

### LC10 正则匹配 (Hard)
**问题**: '.'匹配任意字符，'*'匹配零个或多个前一个字符
**核心**: 动态规划，状态转移复杂

```cpp
dp[i][j] = 
    if p[j-1] == '*':
        dp[i][j] = dp[i][j-2]  // *匹配0次
        if p[j-2] == '.' || p[j-2] == s[i-1]:
            dp[i][j] |= dp[i-1][j]  // *匹配1+次
    else if p[j-1] == '.' || p[j-1] == s[i-1]:
        dp[i][j] = dp[i-1][j-1]
```

**验证**: ✅ 9/9 cases passed

**核心**: dp[i][j] = match?dp[i-1][j-1] : 1+min(删除/插入/替换)

### 并查集模式 (已沉淀)

```cpp
// 路径压缩 + 秩优化
int find(int x) {
    if (parent[x] != x) parent[x] = find(parent[x]);
    return parent[x];
}
void unite(int a, int b) {
    int ra = find(a), rb = find(b);
    if (ra == rb) return;
    if (rank[ra] < rank[rb]) swap(ra, rb);
    parent[rb] = ra;
    if (rank[ra] == rank[rb]) rank[ra]++;
}
```
