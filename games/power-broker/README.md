# 算力经纪人 (Power Broker) v1.4

## 游戏介绍
玩家扮演算力经纪人，在AI芯片短缺时代调配全球算力资源，通过买卖算力赚取利润。

## v1.4丰富化功能 ✅
- 3种难度模式（简单/普通/困难）
- 10种普通事件 + 6种Boss级事件
- 12项成就系统
- AI竞争对手（2-3个AI玩家同步交易）
- 存档/读档系统
- 胜率追踪（连胜系统）

## 运行方法
```bash
cd ~/.openclaw/workspace-developer/games/power-broker
make && ./power-broker
```

## 难度配置
| 难度 | 初始资金 | 波动率 | AI激进度 | 游戏天数 |
|------|----------|--------|----------|----------|
| 简单 | $150,000 | 8% | 30% | 30天 |
| 普通 | $100,000 | 12% | 60% | 30天 |
| 困难 | $60,000 | 18% | 90% | 25天 |

## Boss事件（第10天、第20天）
- 量子计算突破、算力战争爆发、AI觉醒事件
- 算力寒冬、核爆级漏洞、算力之王诞生
- 价格冲击±50%~90%

## 成就系统
| 成就 | 条件 |
|------|------|
| 初入市场 | 完成首次交易 |
| 大额交易 | 单次买卖≥50单位 |
| 小有盈利 | 累计盈利$10,000 |
| 百万富翁 | 累计盈利$100,000 |
| 三连胜 | 连续3天赚钱 |
| 五连胜 | 连续5天赚钱 |
| Boss克星 | 成功应对Boss级事件 |
| 硬核玩家 | 通关Hard难度 |
| 算力大亨 | 持有≥1000单位算力 |
| 孤注一掷 | 全仓买入 |
| 算力交易王 | 总交易量≥1000单位 |
| 触底反弹 | 资产缩水50%后恢复 |

## AI竞争对手
| 名称 | 特点 |
|------|------|
| 算力精灵(AI) | 保守型，稳定小盈 |
| 芯片大亨(AI) | 中等激进 |
| 量子猎手(AI) | 困难模式专属，高风险高收益 |

## 操作
```
[1]买入10  [2]买入50  [3]自定义买入
[4]卖出10  [5]卖出50  [6]自定义卖出
[7]存档    [8]读档    [9]成就列表
[q]退出
```
