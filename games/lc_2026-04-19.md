## 14:45 堆/优先队列专题

### 核心模板

**Kth Largest (最小堆)**：
```python
heap = []
for n in nums:
    heapq.heappush(heap, n)
    if len(heap) > k:
        heapq.heappop(heap)
return heap[0]  # 第k大的数
```

**Top K Frequent (桶排序)**：
```python
freq = [[] for _ in range(len(nums)+1)]
for num, c in Counter(nums).items():
    freq[c].append(num)
# 从高频到低频收集
```

**MedianFinder (双堆)**：
```python
lo = []  # max heap (negate)
hi = []  # min heap
# add: push to lo, balance, move max of lo to hi
# median: lo[0] if len(lo)>len(hi) else avg
```
