# LC贪心算法专题 - 2026-04-17

## 核心思想
每一步都做出局部最优选择，期望达到全局最优。

## 经典问题模式

### 1. 区间调度（Activity Selection）
LC435 无重叠区间 / LC452 用最少数量的箭引爆气球
- 按结束时间排序，选择结束最早且不重叠的
```
sort by end time → greedily select non-overlapping
```

### 2. 跳跃游戏
LC55 跳跃游戏 / LC45 跳跃游戏II
- 维护最远可达位置，当前覆盖范围内最优选择

### 3. 分配问题
LC455 分发饼干 / LC135 分发糖果
- 排序后从最优匹配方向贪心

### 4. 钱币找零
LC860 柠檬水找零
- 优先使用大面额

### 5. 股票买卖
LC122 买卖股票的最佳时机II
- 所有上涨天都买卖 = 全年最大收益

### 6. 会议问题
LC253 会议室II（最少会议室）
- 优先队列按结束时间排序

## 验证框架
```python
def greedy_template(intervals):
    # 1. 排序
    sorted_intervals = sort_by_key(intervals, key='end')
    # 2. 选择
    count = 0
    last_end = -inf
    for interval in sorted_intervals:
        if interval.start >= last_end:
            count += 1
            last_end = interval.end
    return count
```
