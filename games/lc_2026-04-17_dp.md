### 1. LC121 买卖股票最佳时机 (Best Time to Buy and Stock)
**问题**：一次买卖，最大利润
**核心洞察**：记录最低价格，计算最大差值

```python
def maxProfit(prices):
    min_price = float('inf')
    max_profit = 0
    for price in prices:
        max_profit = max(max_profit, price - min_price)
        min_price = min(min_price, price)
    return max_profit

# 验证: [7,1,5,3,6,4]→5(1买6卖)✓ [7,6,4,3,1]→0✓
```

**DP思维**：
- dp[i] = max(dp[i-1], prices[i] - min_price_so_far)
- 空间O(1)，一次遍历

### 2. LC122 买卖股票最佳时机II (Best Time to Buy and Stock II)
**问题**：多次买卖，最大利润
**核心洞察**：累加所有上升段

```python
def maxProfit(prices):
    profit = 0
    for i in range(1, len(prices)):
        if prices[i] > prices[i-1]:
            profit += prices[i] - prices[i-1]
    return profit

# 验证: [7,1,5,3,6,4]→7(1买5卖+3买6卖)✓
```

**核心思想**：利润可以分解为每天的涨跌，累加所有正差值

### 3. LC123 买卖股票最佳时机III (Best Time to Buy and Stock III)
**问题**：最多两次买卖，最大利润
**核心洞察**：5状态DP

```python
def maxProfit(prices):
    if not prices: return 0
    n = len(prices)
    buy1 = sell1 = buy2 = sell2 = float('-inf')
    for price in prices:
        buy1 = max(buy1, -price)
        sell1 = max(sell1, buy1 + price)
        buy2 = max(buy2, sell1 - price)
        sell2 = max(sell2, buy2 + price)
    return max(0, sell2)

# 验证: [3,3,5,0,0,3,1,4]→6✓
```

**状态机**：
- buy1: 第一次买入后的最大收益
- sell1: 第一次卖出后的最大收益
- buy2: 第二次买入后的最大收益
- sell2: 第二次卖出后的最大收益

### 4. LC188 买卖股票最佳时机IV (Best Time to Buy and Stock IV)
**问题**：最多k次买卖
**核心洞察**：通用状态机，buy[i]/sell[i]数组

```python
def maxProfit(k, prices):
    if not prices or k == 0: return 0
    n = len(prices)
    if k >= n // 2:  # 无限次等价
        profit = 0
        for i in range(1, n):
            profit += max(0, prices[i] - prices[i-1])
        return profit
    
    buy = [float('-inf')] * (k + 1)
    sell = [0] * (k + 1)
    for price in prices:
        for i in range(1, k + 1):
            buy[i] = max(buy[i], sell[i-1] - price)
            sell[i] = max(sell[i], buy[i] + price)
    return sell[k]

# 验证: k=2, prices=[2,4,1]→2✓
```

### 5. LC309 最佳买卖股票时机含冷冻期 (Best Time to Buy and Stock with Cooldown)
**问题**：卖出后有一天冷冻期
**核心洞察**：3状态

```python
def maxProfit(prices):
    if not prices: return 0
    n = len(prices)
    sell = 0
    prev_sell = 0
    buy = float('-inf')
    for price in prices:
        prev_sell = sell
        sell = max(sell, buy + price)
        buy = max(buy, prev_sell - price)
    return sell

# 验证: [1,2,3,0,2]→3✓ [1]→0✓
```

**状态转移**：
- sell[i] = max(sell[i-1], buy[i-1] + price)
- buy[i] = max(buy[i-1], prev_sell[i-2] - price)

### 6. LC714 买卖股票最佳时机含手续费 (Best Time to Buy and Stock with Transaction Fee)
**问题**：每次交易有手续费
**核心洞察**：持有vs不持有状态

```python
def maxProfit(prices, fee):
    if not prices: return 0
    cash = 0  # 不持有
    hold = float('-inf')  # 持有
    for price in prices:
        cash = max(cash, hold + price)  # 卖出
        hold = max(hold, cash - price - fee)  # 买入（扣手续费）
    return cash

# 验证: [1,3,2,8,4,9], fee=2 → 8✓
```

### 7. LC300 最长递增子序列 (Longest Increasing Subsequence)
**问题**：找出最长递增子序列长度
**核心洞察**：dp[i] = max(dp[j] + 1) for j < i

```python
def lengthOfLIS(nums):
    if not nums: return 0
    n = len(nums)
    dp = [1] * n
    for i in range(n):
        for j in range(i):
            if nums[j] < nums[i]:
                dp[i] = max(dp[i], dp[j] + 1)
    return max(dp)

# 验证: [10,9,2,5,3,7,101,18]→4([2,3,7,101])✓
```

**二分优化**：O(n log n)，维护有序数组

### 8. LC1143 最长公共子序列 (Longest Common Subsequence)
**问题**：两个字符串的LCS长度
**核心洞察**：二维dp

```python
def longestCommonSubsequence(text1, text2):
    m, n = len(text1), len(text2)
    dp = [[0] * (n + 1) for _ in range(m + 1)]
    for i in range(1, m + 1):
        for j in range(1, n + 1):
            if text1[i-1] == text2[j-1]:
                dp[i][j] = dp[i-1][j-1] + 1
            else:
                dp[i][j] = max(dp[i-1][j], dp[i][j-1])
    return dp[m][n]

# 验证: "abcde"+"ace"→3("ace")✓ "abc"+"abc"→3✓
```

### 9. LC72 编辑距离 (Edit Distance)
**问题**：word1转word2最少操作数
**核心洞察**：插入/删除/替换三操作

```python
def minDistance(word1, word2):
    m, n = len(word1), len(word2)
    dp = [[0] * (n + 1) for _ in range(m + 1)]
    for i in range(m + 1): dp[i][0] = i
    for j in range(n + 1): dp[0][j] = j
    for i in range(1, m + 1):
        for j in range(1, n + 1):
            if word1[i-1] == word2[j-1]:
                dp[i][j] = dp[i-1][j-1]
            else:
                dp[i][j] = 1 + min(dp[i-1][j], dp[i][j-1], dp[i-1][j-1])
    return dp[m][n]

# 验证: "horse"+"ros"→3✓ "intention"+"execution"→5✓
```

### 10. LC322 零钱兑换 (Coin Change)
**问题**：凑成amount最少硬币数
**核心洞察**：完全背包，dp[i] = min(dp[i], dp[i-coin] + 1)

```python
def coinChange(coins, amount):
    dp = [float('inf')] * (amount + 1)
    dp[0] = 0
    for coin in coins:
        for i in range(coin, amount + 1):
            dp[i] = min(dp[i], dp[i - coin] + 1)
    return dp[amount] if dp[amount] != float('inf') else -1

# 验证: coins=[1,2,5], amount=11→3(5+5+1)✓
```
