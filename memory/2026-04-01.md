## 经验沉淀

### 技术难点1: WASM ccall函数
- 改用`Reflect.apply`替代复杂的`Function.apply`动态代码生成
- WebAssembly.instantiate直接加载.wasm文件

### 技术难点2: Sliding Window变长窗口
- `if (seen[c] >= left) left = seen[c] + 1` 收缩左边界
- 时间O(n),空间O(min(charset_size, n))

### 技术难点3: BST严格边界
- 必须用long long处理INT_MIN/INT_MAX
- `node->val <= minVal || node->val >= maxVal` → false

### 技术难点4: Snake方向锁
- dir_timer=3帧锁定期防止一帧内多次改向导致180°翻转

### 技术难点5: Backtracking括号生成
- Catalan数: C_n = (2n)!/(n!(n+1)!)
- n=3→5, n=4→14, n=5→42
