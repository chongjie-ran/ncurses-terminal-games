### LC146 LRU Cache — LRU 缓存 ✅

**题目**: 实现 O(1) get 和 put 的 LRU 缓存  
**核心算法**: 哈希表 + 双向链表

**核心思路**:
- `unordered_map<key, list::iterator>` 快速定位
- `list<pair<key,value>>` 维护顺序：前端 = 最近使用 (MRU)，后端 = 最久未使用 (LRU)
- `get(key)`: 找到后 splice 移动到前端，返回值
- `put(key, value)`: 找到则更新并移动到前端；未找到则插入前端，超容量则删除后端

**代码实现**:
```cpp
list<pair<int,int>> lst;                                    // front=MRU, back=LRU
unordered_map<int, list<pair<int,int>>::iterator> mp;

int get(int key) {
    auto it = mp.find(key);
    if (it == mp.end()) return -1;
    lst.splice(lst.begin(), lst, it->second);  // move to front
    return it->second->second;
}

void put(int key, int value) {
    if (mp.count(key)) {
        mp[key]->second = value;
        lst.splice(lst.begin(), lst, mp[key]);
    } else {
        if (lst.size() == cap) { auto last = lst.back(); mp.erase(last.first); lst.pop_back(); }
        lst.emplace_front(key, value);
        mp[key] = lst.begin();
    }
}
```

**验证**: ✅ LeetCode 测试用例全部通过

### LC295 Find Median from Data Stream — 数据流中位数 ✅

**题目**: 数据流动态加入，随时查询中位数（O(log n) add, O(1) median）  
**核心算法**: 双堆（max-heap + min-heap）

**核心思路**:
- `maxHeap`: 存储较小的一半（max-heap），顶部 = 最大值
- `minHeap`: 存储较大的一半（min-heap），顶部 = 最小值
- 不变量：`maxHeap.size() == minHeap.size()` 或 `maxHeap.size() == minHeap.size() + 1`
- 中位数：`maxHeap.size() > minHeap.size()` ? `maxHeap.top()` : `(maxHeap.top() + minHeap.top()) / 2.0`
- 新元素：根据与 maxHeap.top() 比较决定加入哪个堆，然后 balance

**关键洞察（调试Bug修复）**:
- 原错误：用 `maxHeap.push(-num)` 实现 max-heap（存储负数）
- 问题：移动元素时容易混淆正负，导致中位数错误
- 正确方法：直接用 `priority_queue<int>`（max-heap存正数），`priority_queue<int, vector<int>, greater<int>>`（min-heap）
- 平衡时：`maxHeap > minHeap + 1` → 移动 maxHeap.top() 到 minHeap；`maxHeap < minHeap` → 移动 minHeap.top() 到 maxHeap

**代码实现**:
```cpp
priority_queue<int> maxHeap;                                       // lower half
priority_queue<int, vector<int>, greater<int>> minHeap;             // upper half

void addNum(int num) {
    if (maxHeap.empty() || num <= maxHeap.top()) maxHeap.push(num);
    else minHeap.push(num);

    if (maxHeap.size() > minHeap.size() + 1) {
        minHeap.push(maxHeap.top()); maxHeap.pop();  // move largest lower to upper
    } else if (maxHeap.size() < minHeap.size()) {
        maxHeap.push(minHeap.top()); minHeap.pop();  // move smallest upper to lower
    }
}

double findMedian() {
    if (maxHeap.size() > minHeap.size()) return (double)maxHeap.top();
    else return (double)(maxHeap.top() + minHeap.top()) / 2.0;
}
```

**验证**: ✅ [1,2]→1.5, [1,2,3]→2.0, [1,2,3,4]→2.5, 负数测试全部通过

| 驱逐 | 删除 lst.back() + mp.erase | O(1) |

**双向链表操作**:
- `splice(pos, lst, it)`: 将 it 从 lst 移动到 pos 位置（O(1)）
- 配合 unordered_map<key, list::iterator> 实现 O(1) 随机访问 + O(1) 移动

#### 双堆求中位数设计要点

| 堆 | 类型 | 存储 | 作用 |
|----|------|------|------|
| maxHeap | max-heap | 较小的一半 | 下半部分最大值 |
| minHeap | min-heap | 较大的一半 | 上半部分最小值 |

**平衡操作**:
```cpp
if (maxHeap.size() > minHeap.size() + 1)
    minHeap.push(maxHeap.top()), maxHeap.pop();
else if (maxHeap.size() < minHeap.size())
    maxHeap.push(minHeap.top()), minHeap.pop();
```

**中位数公式**:
```cpp
maxHeap.size() > minHeap.size() ? maxHeap.top() : (maxHeap.top() + minHeap.top()) / 2.0
```

**调试Bug教训**: 使用负数存储 max-heap 容易在 rebalance 时混淆正负。正确做法是用 `priority_queue<int>`（默认max-heap）直接存储正数，不需要取负。
