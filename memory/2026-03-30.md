### Raylib 绘图模式

1. **3D凸起格子**（隐藏状态）:
   ```c
   DrawRectangle(x, y, s, s, CLR_CELL_HIDDEN);
   DrawRectangle(x, y, s, 3, LIGHT_BEVEL);   // top
   DrawRectangle(x, y, 3, s, LIGHT_BEVEL);    // left
   DrawRectangle(x, y+s-3, s, 3, DARK_BEVEL); // bottom
   DrawRectangle(x+s-3, y, 3, s, DARK_BEVEL); // right
   ```

2. **3D凹陷格子**（翻开状态）:
   ```c
   DrawRectangle(x, y, s, s, CLR_CELL_REVEAL);
   DrawRectangle(x, y, s, 2, DARKER);  // top → darker (light from top-left)
   DrawRectangle(x, y, 2, s, DARKER);  // left
   ```

3. **地雷**:
   ```c
   DrawCircle(cx, cy, r, MINE_COL);           // body
   DrawCircle(cx, cy, r*0.6, BRIGHT_COL);     // highlight
   for (int a=0; a<8; a++) {                   // spokes
       int ex = cx + cos(a*PI/4)*r*1.3;
       int ey = cy + sin(a*PI/4)*r*1.3;
       DrawLine(cx, cy, ex, ey, MINE_COL);
   }
   ```

4. **旗帜**:
   ```c
   DrawLine(px, top, px, bot, GRAY_POLE);
   DrawTriangle((Vector2){px, top},
                (Vector2){px+10, top+6},
                (Vector2){px, top+12}, FLAG_COL);
   ```

```c
    // 扩展右指针
```

### Topological Sort (Kahn) 模板

```c
// Kahn算法 - BFS拓扑排序
vector<vector<int>> g(n);
vector<int> indeg(n, 0);
for (auto& e : edges) { g[e[0]].push_back(e[1]); indeg[e[1]]++; }

queue<int> q;
for (int i = 0; i < n; i++) if (indeg[i] == 0) q.push(i);

vector<int> order;
while (!q.empty()) {
    int u = q.front(); q.pop();
    order.push_back(u);
    for (int v : g[u])
        if (--indeg[v] == 0) q.push(v);
}
bool cycle = (order.size() != n); // 有环
```

**LC85 Maximal Rectangle — 最大矩形** ✅

**题目**: 2D二进制矩阵中找全1的最大矩形面积  
**核心算法**: 单调栈（逐行处理，将每列高度视为柱状图高度）

**核心思路**:
- 维护每列的连续1高度（从顶部到当前行）
- 将每行视为柱状图，用单调栈求最大矩形
- 时间 O(mn)，空间 O(n)

**代码实现**:
```cpp
int maximalRectangle(vector<vector<char>>& matrix) {
    int m = matrix.size(), n = matrix[0].size();
    vector<int> left(n, 0), right(n, n), height(n, 0);
    int best = 0;
    for (int i = 0; i < m; ++i) {
        int cur_left = 0, cur_right = n;
        for (int j = 0; j < n; ++j)
            height[j] = (matrix[i][j]=='1') ? height[j]+1 : 0;
        for (int j = 0; j < n; ++j)
            if (matrix[i][j]=='1') left[j]=max(left[j],cur_left);
            else { left[j]=0; cur_left=j+1; }
        for (int j = n-1; j >=0; --j)
            if (matrix[i][j]=='1') right[j]=min(right[j],cur_right);
            else { right[j]=n; cur_right=j; }
        for (int j = 0; j < n; ++j)
            best = max(best, (right[j]-left[j])*height[j]);
    }
    return best;
}
```

**验证**: ✅ m1=6, m2=0, m3=1 全部通过

**LC115 Distinct Subsequences — 不同的子序列** ✅

**题目**: s中有多少种方式可以得到t？  
**核心算法**: 1D DP（空间优化版）

**核心思路**:
- `dp[j]` = 用 s[0..i] 生成 t[0..j-1] 的方式数
- 1D 优化：`dp[j] += dp[j-1]` 当 `s[i-1] == t[j-1]`
- 从后往前遍历 j 避免覆盖
- 时间 O(mn)，空间 O(n)

**代码实现**:
```cpp
int numDistinct(string s, string t) {
    int m = s.size(), n = t.size();
    vector<long long> dp(n + 1, 0);
    dp[0] = 1;
    for (int i = 1; i <= m; ++i)
        for (int j = n; j >= 1; --j)
            if (s[i-1] == t[j-1]) dp[j] += dp[j-1];
    return (int)dp[n];
}
```

**验证**: ✅ rabbbit/rabbit=3, babgbag/bag=5, ""/"a"=0, "a"/"a"=1 全部通过

**LC221 Maximal Square — 最大正方形** ✅

**题目**: 二进制矩阵中找全1的最大正方形面积  
**核心算法**: DP + 空间优化（O(n)）

**核心思路**:
- `dp[i][j]` = 以 (i-1,j-1) 为右下角的最大正方形边长
- `dp[i][j] = min(dp[i-1][j], dp[i][j-1], dp[i-1][j-1]) + 1` 当 matrix[i-1][j-1]=='1'
- 1D 优化：`dp[j] = min(dp[j], dp[j-1], prev) + 1`，用 prev 记录 dp[j-1] 旧值
- `prev` 每行开始要重置为 0

**代码实现**:
```cpp
int maximalSquare(vector<vector<char>>& matrix) {
    int m = matrix.size(), n = matrix[0].size();
    vector<int> dp(n + 1, 0);
    int maxSide = 0, prev = 0;
    for (int i = 1; i <= m; ++i) {
        for (int j = 1; j <= n; ++j) {
            int temp = dp[j];
            if (matrix[i-1][j-1] == '1') {
                dp[j] = min({dp[j], dp[j-1], prev}) + 1;
                maxSide = max(maxSide, dp[j]);
            } else {
                dp[j] = 0;
            }
            prev = temp;
        }
        prev = 0;
    }
    return maxSide * maxSide;
}
```

**验证**: ✅ m1=4(2x2), m2=0, m3=1 全部通过

### 技术沉淀

#### 1D DP 空间优化的 prev 变量模式

当从 2D DP 压缩到 1D 时，如果递推涉及 `dp[i-1][j-1]`，需要 `prev` 变量记录上一轮迭代的值：

```cpp
// 2D: dp[i][j] = min(dp[i-1][j], dp[i][j-1], dp[i-1][j-1]) + 1
// 1D:
int prev = 0;
for (int i = 1; i <= m; ++i) {
    for (int j = 1; j <= n; ++j) {
        int temp = dp[j];          // 记住旧值（dp[j-1]的当前行值）
        if (matrix[i-1][j-1] == '1') {
            dp[j] = min({dp[j], dp[j-1], prev}) + 1;
        } else {
            dp[j] = 0;
        }
        prev = temp;               // 更新 prev 为旧值
    }
    prev = 0;                      // 每行开始重置为 0
}
```

**关键**: `prev` 在每行开始时必须重置为 0，因为第一列没有左上角邻居。

#### LC85 最大矩形 vs LC221 最大正方形

| 问题 | 核心算法 | 关键差异 |
|------|---------|---------|
| LC85 最大矩形 | 单调栈逐行 | 求任意形状矩形，面积=宽×高 |
| LC221 最大正方形 | DP | 只求正方形，边长=min(左/上/左上)+1 |

