**LC139 Word Break — 单词拆分** ✅

**题目**: 判断字符串能否由字典中的单词空格拼接组成  
**核心算法**: 1D DP

**核心思路**:
- `dp[i]` = s[0..i-1] 是否可以被拆分
- 对每个位置 j，检测 s[j..i-1] 是否在字典中
- 时间 O(n² × word_len)，空间 O(n)

**代码实现**:
```cpp
bool wordBreak(string s, vector<string>& wordDict) {
    unordered_set<string> dict(wordDict.begin(), wordDict.end());
    int n = s.size();
    vector<bool> dp(n+1, false);
    dp[0] = true;
    for (int i = 1; i <= n; ++i) {
        for (int j = 0; j < i; ++j) {
            if (dp[j] && dict.count(s.substr(j, i-j))) {
                dp[i] = true;
                break;
            }
        }
    }
    return dp[n];
}
```

**LC1143 Longest Common Subsequence — 最长公共子序列** ✅

**题目**: 两个字符串的最长公共子序列长度  
**核心算法**: 1D DP + prev 变量（空间优化）

**核心思路**:
- `dp[j]` = text1[0..i-1] 与 text2[0..j-1] 的 LCS 长度
- 当 `text1[i-1] == text2[j-1]` 时，`dp[j] = prev + 1`
- 否则 `dp[j] = max(dp[j], dp[j-1])`
- `prev` 保存 `dp[j-1]` 旧值（当前行左上角）

**代码实现**:
```cpp
int longestCommonSubseq(string text1, string text2) {
    int m = text1.size(), n = text2.size();
    vector<int> dp(n+1, 0);
    for (int i = 1; i <= m; ++i) {
        int prev = 0;
        for (int j = 1; j <= n; ++j) {
            int temp = dp[j];
            if (text1[i-1] == text2[j-1])
                dp[j] = prev + 1;
            else
                dp[j] = max(dp[j], dp[j-1]);
            prev = temp;
        }
    }
    return dp[n];
}
```

**LC514 Freedom Trail — 自由之路** ✅

**题目**: 环形转盘上拼写单词的最小步数  
**核心算法**: DP + 贪心选择（Hard）

**核心思路**:
- 转盘长度为 n，每次可顺时针或逆时针转到目标字母
- `dp[i][j]` = 拼到 key[i] 且当前在 ring 位置 j 的最小步数
- 对每个 key 字符，找出 ring 中所有该字符位置，贪心选最小代价
- 步数 = 顺/逆时针最小距离 + 1（按确认键）
- 时间 O(m × n²)，空间 O(n)

**代码实现**:
```cpp
int findRotateSteps(string ring, string key) {
    int n = ring.size(), m = key.size();
    unordered_map<char, vector<int>> pos;
    for (int i = 0; i < n; ++i) pos[ring[i]].push_back(i);
    vector<int> dp(n, INT_MAX);
    dp[0] = 0;
    for (int i = 0; i < m; ++i) {
        vector<int> ndp(n, INT_MAX);
        for (int rpos = 0; rpos < n; ++rpos) {
            if (dp[rpos] == INT_MAX) continue;
            for (int p : pos[key[i]]) {
                int diff = abs(rpos - p);
                int step = min(diff, n - diff);
                ndp[p] = min(ndp[p], dp[rpos] + step + 1);
            }
        }
        dp.swap(ndp);
    }
    return *min_element(dp.begin(), dp.end());
}
```

**LC416 Partition Equal Subset Sum — 分割等和子集** ✅

**题目**: 能否将数组分成两部分，和相等？  
**核心算法**: 0/1 背包 DP

**核心思路**:
- 总和为奇数 → 不可分割
- target = sum / 2
- `dp[j]` = 是否可以用某些数字填满容量 j
- 从后往前遍历防止重复使用
- 时间 O(n × target)，空间 O(target)

**代码实现**:
```cpp
bool canPartition(vector<int>& nums) {
    int sum = 0;
    for (int x : nums) sum += x;
    if (sum % 2) return false;
    int target = sum / 2;
    vector<bool> dp(target + 1, false);
    dp[0] = true;
    for (int num : nums) {
        for (int j = target; j >= num; --j) {
            dp[j] = dp[j] || dp[j - num];
        }
    }
    return dp[target];
}
```

**LC474 Ones and Zeroes — 1和0** ✅

**题目**: 最多选多少个字符串，使得0不超过m个、1不超过n个  
**核心算法**: 二维 0/1 背包 DP

**核心思路**:
- 2维背包：m 个 0，n 个 1
- `dp[i][j]` = 最多选多少字符串满足 i 个 0、j 个 1
- 从后往前遍历两个维度
- 时间 O(S × m × n)，空间 O(m × n)

**代码实现**:
```cpp
int findMaxForm(vector<string>& strs, int m, int n) {
    vector<vector<int>> dp(m+1, vector<int>(n+1, 0));
    for (string& s : strs) {
        int cnt0 = 0, cnt1 = 0;
        for (char c : s) if (c=='0') cnt0++; else cnt1++;
        for (int i = m; i >= cnt0; --i)
            for (int j = n; j >= cnt1; --j)
                dp[i][j] = max(dp[i][j], dp[i-cnt0][j-cnt1] + 1);
    }
    return dp[m][n];
}
```

| 问题 | 维度 | 核心递推 |
|------|------|---------|

int prev = 0;

### 经验沉淀

#### 背包 DP 通用模板

```cpp
// 1D 背包（每种物品选一次）
for (int item : items) {
    for (int j = capacity; j >= item; --j)
        dp[j] = max(dp[j], dp[j - item] + value);
}

// 2D 背包（两种限制）
for (string& s : strs) {
    int cnt0 = 0, cnt1 = 0;
    for (char c : s) c=='0' ? cnt0++ : cnt1++;
    for (int i = m; i >= cnt0; --i)
        for (int j = n; j >= cnt1; --j)
            dp[i][j] = max(dp[i][j], dp[i-cnt0][j-cnt1] + 1);
}
```

#### LCS 的 prev 变量精髓

当 2D DP 状态依赖 `dp[i-1][j-1]`（左上角）时：
- 需要 `prev` 记住 `dp[j-1]` 旧值
- `prev` 在每轮 j 循环开始时更新为 `dp[j]` 旧值
- `dp[j] = prev + 1` 当匹配时
