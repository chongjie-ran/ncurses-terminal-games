## 网络问题记录

### GitHub HTTPS (443) 连接问题
- **现象**：`git push` via HTTPS 失败，HTTP2 framing layer error
- **原因**：网络环境对GitHub 443端口有限制
- **解决**：使用SSH remote (`git@github.com:...`) 替代HTTPS
