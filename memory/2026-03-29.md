**LC315 Count of Smaller Numbers After Self** ✅

**核心算法**: 树状数组 BIT + 离散化

**思路**:
- 从右往左扫描
- 对每个数，查询 BIT 中比它小的数的个数（`sum(idx-1)`）
- 然后将该数插入 BIT（`add(idx, 1)`）
- 离散化处理负数和重复值

```cpp
// 核心：BIT 的 sum 操作求"有多少数比 nums[i] 小"
res[i] = bit.sum(idx - 1);
bit.add(idx, 1);
```

**时间复杂度**: O(n log n)，空间 O(n)

---

**LC218 The Skyline Problem** ✅

**核心算法**: 扫描线 + 事件排序 + 主动删除式堆

**思路**:
1. 构建事件：`(x, height, isStart)`，起点优先
2. 按 x 排序事件
3. 处理每个 x：
   - 先删所有该 x 的 end 事件
   - 再加所有该 x 的 start 事件
   - 当前最大高度 = 活跃建筑最高点
   - 若高度变化，输出关键点 `(x, height)`

**主动删除法**（不用 lazy deletion）:
- 用 `map<int,int>` 记录活跃高度及计数
- start: `active[h]++`
- end: `active[h]--` 若为0则 `erase(h)`
- 当前最大 = `active.rbegin()->first`

**关键洞察**:
- 为什么主动删除？因为同一高度可能有多个建筑
- `map` 的 `rbegin()` 即最大键

**时间复杂度**: O(n log n)，空间 O(n)
