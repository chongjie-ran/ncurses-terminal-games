#### 技术难点 & 解决方案
- **Grid indexing**: 使用 idx = row * GRID_W + col 计算一维数组索引
- **Click detection**: 根据canvas内点击位置反推card index
- **WAITING state**: 防止匹配失败时用户继续点击，等待timer翻转回
- **Deterministic shuffle**: 用 `(i*7+3) % (i+1)` 实现无随机数的Fisher-Yates
