## 技术沉淀

### LC239 单调队列模板

```cpp
deque<int> dq; // 存索引，值单调递减
for (int i = 0; i < n; i++) {
    // 1. 移除队尾小于当前元素的（永远不可能成为最大值）
    while (!dq.empty() && nums[dq.back()] <= nums[i])
        dq.pop_back();
    dq.push_back(i);

    // 2. 移除队首超出窗口的
    if (dq.front() <= i - k) dq.pop_front();

    // 3. 记录答案
    if (i >= k - 1) ans.push_back(nums[dq.front()]);
}
```

### Emscripten WASM 编译步骤

```bash
source ~/emsdk/emsdk_env.sh

emcc \
  game.c wasm_renderer.c \
  -o game.js \
  -s WASM=1 \
  -s EXPORTED_FUNCTIONS='["_init_game","_update_game",...]' \
  -s EXPORTED_RUNTIME_METHODS='["ccall","cwrap"]' \
  -s ALLOW_MEMORY_GROWTH=1 \
  -s MODULARIZE=1 \
  -s EXPORT_NAME="GameModule" \
  --no-entry -O2
```

### WASM + Canvas 渲染架构

```javascript
// 1. 加载 WASM 模块
var Module = {
  onRuntimeInitialized: function() { initGame(); }
};

// 2. 调用 C 函数
Module.ccall('init_game', null, [], []);
Module._update_game(dt);  // 直接调用

// 3. 读取游戏状态
const frogX = Module._get_frog_x();
const lives = Module._get_frog_lives();

// 4. Canvas 2D 渲染
ctx.fillStyle = '#4ade80';
ctx.beginPath();
ctx.arc(frogX + CELL/2, frogY + CELL/2, CELL*0.4, 0, Math.PI*2);
ctx.fill();
```
