## 技术积累

### Trie 前缀树 (LC208/LC212)
- insert/search/startsWith: O(L) per op
- LC212: Trie + DFS 批量查询 (组合LC79+LC208)
- 关键: TrieNode存储 `string word` 避免重建

### Kth Largest (LC215)
- 最小堆O(n log k): 维护大小K的最小堆
- 快速选择O(n): 分治思想，partition返回pivot位置
- 关键洞察: 第K大 = index n-k (升序)

### Union-Find (LC547)
- 路径压缩 + 按秩合并: O(α(n)) ≈ O(1)
- 对称矩阵只遍历 j>i 半区
